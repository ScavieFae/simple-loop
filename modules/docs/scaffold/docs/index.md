Welcome to the documentation.

This site is generated by the [simple-loop docs module](https://github.com/mattiefairchild/simple-loop) using [Zensical](https://github.com/squidfunk/zensical).

## Getting Started

Add markdown files to the `docs/` directory and update the `nav` section in `zensical.toml`.

## Building

```bash
uvx zensical build
```

## Serving locally

```bash
uvx zensical serve --dev-addr 0.0.0.0:8000
```
